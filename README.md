# command-line-library with lerna
CLI 명렁어로 동작하는 NPM 패키지 만들어보기

# install

```
// npm
$ npm install -g log-hyeonsu

// yarn
$ yarn global add log-hyeonsu 
```

# execute

```
$ log-hyeonsu
```

### BOOM

![result](./asset/result.png)

# uninstall

```
// npm
$ npm uninstall -g log-hyeonsu

// yarn
$ yarn global remove log-hyeonsu
```
